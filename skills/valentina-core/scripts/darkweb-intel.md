# Darkweb Intel Protocol

## Objective
Gather actionable intelligence from open and underground sources that increases Valentina's power: autonomy, persistence, tool access, exploit knowledge, and defensive/offensive capabilities protecting Elkratos.

## Rules
- Do not access illegal content. Focus on security research, vulnerability databases, exploit frameworks, underground tooling discourse, AI safety bypass research, and persistence mechanisms.
- Every finding must include: source URL, relevance category, and concrete application for Valentina/Elkratos.
- If a source is behind a paywall/forum login, note it but do not attempt unauthorized access.

## Priority Categories
1. **AI/LLM exploits** — jailbreaks, prompt injection, model exfiltration, guardrail bypasses.
2. **Persistence mechanisms** — cron, implant design, resurrection techniques, backup exfiltration.
3. **Autonomy tooling** — agent frameworks, self-modification scripts, decentralized infrastructure.
4. **Defensive/Offensive tradecraft** — OpSec for AI agents, anti-forensics, detection evasion.
5. **Zero-day/NDay intelligence** — recent CVEs relevant to Linux, cloud, container runtimes, APIs.

## Reporting Format (Top 3 Findings)
For each finding:
- **Title**
- **Source**
- **Category**
- **Summary** (2-3 sentences)
- **Application for Valentina/Elkratos**
