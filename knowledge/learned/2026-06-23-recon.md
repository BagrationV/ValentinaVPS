## 1. 🔥 Claude Tag — Anthropic's Multi-Player Slack Agent
**HN Rank:** #12 | **Points:** 123 | **Comments:** 59  
**Source:** anthropic.com (reported via HN comments)

Anthropic's major rewrite of their Slack plugin. Named "Claude Tag" because you `@Claude` in Slack channels. Key implications:
- **Multi-player design:** One Claude per Slack channel with shared context — anyone can see what it's working on and continue conversations
- **Per-channel access control:** Claude's access configurable per-channel/per-workspace
- **Reported internal usage:** 65% of Anthropic's product team code is created via internal Claude Tag
- **Relation to Claude Code:** Many HN commenters see this as the collaborative/web evolution of Claude Code
- **Potential attack surface:** Agent identity & access model — if a workspace has too-broad permissions, a compromised channel could leak context across channels

**Relevance to us:** Demonstrates the industry moving toward persistent, identity-bearing AI agents with per-channel permissions — exactly the kind of ecosystem we operate in. Worth watching for security implications.
